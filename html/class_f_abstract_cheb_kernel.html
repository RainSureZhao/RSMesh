template<class FReal, class CellClass, class ContainerClass, class MatrixKernelClass, int ORDER, int NVALS = 1>
class FAbstractChebKernel< FReal, CellClass, ContainerClass, MatrixKernelClass, ORDER, NVALS >

This kernels implement the Chebyshev interpolation based FMM operators. It implements all interfaces (P2P, P2M, M2M, M2L, L2L, L2P) which are required by the FFmmAlgorithm and FFmmAlgorithmThread.

作者
Matthias Messner(matth.nosp@m.ias..nosp@m.messn.nosp@m.er@i.nosp@m.nria..nosp@m.fr)
模板参数
CellClassType of cell
ContainerClassType of container to store particles
MatrixKernelClassType of matrix kernel function
ORDERChebyshev interpolation order
