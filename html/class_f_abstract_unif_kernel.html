template<class FReal, class CellClass, class ContainerClass, class MatrixKernelClass, int ORDER, int NVALS = 1>
class FAbstractUnifKernel< FReal, CellClass, ContainerClass, MatrixKernelClass, ORDER, NVALS >

This kernels implement the Lagrange interpolation based FMM operators. It implements all interfaces (P2P, P2M, M2M, M2L, L2L, L2P) which are required by the FFmmAlgorithm and FFmmAlgorithmThread.

作者
Pierre Blanchard (pierr.nosp@m.e.bl.nosp@m.ancha.nosp@m.rd@i.nosp@m.nria..nosp@m.fr)
模板参数
CellClassType of cell
ContainerClassType of container to store particles
MatrixKernelClassType of matrix kernel function
ORDERLagrange interpolation order

Related publications: Fast hierarchical algorithms for generating Gaussian random fields (https://hal.inria.fr/hal-01228519)
