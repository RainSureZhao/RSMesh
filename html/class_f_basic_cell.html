详细描述

This class is used in the FTestKernels, please look at this class to know how to customize a cell.

作者
Berenger Bramas (beren.nosp@m.ger..nosp@m.brama.nosp@m.s@in.nosp@m.ria.f.nosp@m.r)

This class defines a basic cell used for examples. It extends the mininum, only what is needed by FOctree and FFmmAlgorithm to make the things working. By using this extension it will implement the FAbstractCell without inheriting from it.

作者
Berenger Bramas (beren.nosp@m.ger..nosp@m.brama.nosp@m.s@in.nosp@m.ria.f.nosp@m.r)

This cell simply store the data when up/down. It also shows how to be restored and saved, etc.
