作者
Matthias Messner(matth.nosp@m.ias..nosp@m.messn.nosp@m.er@i.nosp@m.nria..nosp@m.fr)

This kernels implement the Chebyshev interpolation based FMM operators exploiting the symmetries in the far-field. It implements all interfaces (P2P, P2M, M2M, M2L, L2L, L2P) which are required by the FFmmAlgorithm and FFmmAlgorithmThread.

模板参数
CellClassType of cell
ContainerClassType of container to store particles
MatrixKernelClassType of matrix kernel function
ORDERChebyshev interpolation order
