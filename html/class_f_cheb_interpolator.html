template<class FReal, int ORDER, class MatrixKernelClass = struct FInterpMatrixKernelR<FReal>, int NVALS = 1>
class FChebInterpolator< FReal, ORDER, MatrixKernelClass, NVALS >
作者
Matthias Messner (matth.nosp@m.ias..nosp@m.matth.nosp@m.ias@.nosp@m.inria.nosp@m..fr) Please read the license

The class FChebInterpolator defines the anterpolation (M2M) and interpolation (L2L) concerning operations.

PB: MatrixKernelClass is passed as template in order to inform interpolator of the size of the vectorial interpolators. Default is the scalar matrix kernel class of type ONE_OVER_R (NRHS=NLHS=1).
