template<class FReal, class CellClass, class ContainerClass, class MatrixKernelClass, int NVALS = 1>
class FCutOffKernel< FReal, CellClass, ContainerClass, MatrixKernelClass, NVALS >

A small kernel to perform only P2P with aMatrixKernel

作者
Olivier Coulaud (Olivi.nosp@m.er.c.nosp@m.oulau.nosp@m.d@in.nosp@m.ria.f.nosp@m.r)

This kernels implement the Lagrange interpolation based FMM operators. It implements all interfaces (P2P,P2M,M2M,M2L,L2L,L2P) which are required by the FFmmAlgorithm and FFmmAlgorithmThread.

模板参数
CellClassType of cell
ContainerClassType of container to store particles
MatrixKernelClassType of matrix kernel function
