Berenger Bramas (beren.nosp@m.ger..nosp@m.brama.nosp@m.s@in.nosp@m.ria.f.nosp@m.r)

This class is an extenssion. It proposes a target/source extenssion for cell. Because cells may have child that contains only sources or targets (in Tsm system) then it is important to not compute for nothing.
