Berenger Bramas (beren.nosp@m.ger..nosp@m.brama.nosp@m.s@in.nosp@m.ria.f.nosp@m.r)

This class is a basic FMM algorithm with periodic behavior It just iterates on a tree and call the kernels with good arguments.

Of course this class does not deallocate pointer given in arguments.
