class FFmmAlgorithmThread< OctreeClass, CellClass, ContainerClass, KernelClass, LeafClass, P2PExclusionClass >

Implements an FMM algorithm threaded using OpenMP.

作者
Berenger Bramas (beren.nosp@m.ger..nosp@m.brama.nosp@m.s@in.nosp@m.ria.f.nosp@m.r)

Please read the license

This class runs a threaded FMM algorithm. It just iterates on a tree and call the kernels with good arguments. The inspector-executor model is used : the class iterates on the tree and builds an array and works in parallel on this array.

When using this algorithm the P2P is thread safe.

This class does not deallocate pointers given to its constructor.
