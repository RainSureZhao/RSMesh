详细描述

template<class FReal, unsigned NbSymbAttributes, unsigned NbAttributesPerParticle, class AttributeClass = FReal>
class FGroupAttachedLeaf< FReal, NbSymbAttributes, NbAttributesPerParticle, AttributeClass >

This class is "attached" to a buffer. It is a wrapper/interface to a memory area that stores all the particles info. The idea is to hidde the group allocation from the group tree but to keep the same interface than the FBasicParticlesContainer.
