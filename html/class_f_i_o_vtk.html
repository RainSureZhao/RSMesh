Print the graph in a VTK file

The output file name is [VTKName][ifile].vtk where ndigits specifies the number of characters to be written for [ifile].

We save in VTK 3.0 format and in ascii mode. See for more details on this format

http://www.cacr.caltech.edu/~slombey/asci/vtk/vtk_formats.simple.html

The data saved are the following.

  • On Nodes: velocitie and force.
  • On segments: length, burgers vector and morton index.
参数
VTKnamefilename prefix
ndigitsthe number of digits to be written
ifilefile number
titledescription of the vtk file
