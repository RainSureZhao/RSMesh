MPI context management

作者
Berenger Bramas (beren.nosp@m.ger..nosp@m.brama.nosp@m.s@in.nosp@m.ria.f.nosp@m.r)

This class manages the initialization and destruction of the MPI context. It also gives access to the main communicator via its const FComm& global() member.

警告
Do not create more that one instance of this class for the whole program lifetime. MPI does not support multiple initializations.

Please read the license.
