This class is an octree container.

Please refere to testOctree.cpp to see an example.

// can be used as : <br>
FOctree<TestParticle, TestCell> tree(1.0,FPoint<FReal>(0.5,0.5,0.5));
FOctree
Definition FOctree.hpp:42
FPoint
Definition FPoint.hpp:26

Particles and cells has to respect the Abstract class definition. Particle must extend {FExtendPosition} Cell must extend extend proposes accessors to FTreeCoordinate and MortonIndex.

If the octree as an height H, then it goes from 0 to H-1 at level 0 the space is not split CellAllocator can be FListBlockAllocator<CellClass, 10> or FBasicBlockAllocator<CellClass>
