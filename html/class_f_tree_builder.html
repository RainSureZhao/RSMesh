Please read the license

作者
Cyrille Piacibello, Berenger Bramas

This class provides a way to insert efficiently large amount of particles inside a tree.

This is a static class. It's useless to instance it. This class use the Threaded QuickSort or the output of FMpiTreeBuilder in order to sort the parts and insert them.
