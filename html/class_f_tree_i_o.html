template<class FReal>
class FTreeIO< FReal >

This class proposes static methods to save and load a tree. It used binary format (so FReal must be the same!)

The format is : [nb leaves] [nb particles in leaf][particles.....] ... [nb cells at level] [morton index][cell] .... ...
