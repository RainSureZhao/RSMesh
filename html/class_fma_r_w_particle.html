template<class FReal, unsigned int READ, unsigned int WRITE>
class FmaRWParticle< FReal, READ, WRITE >

Particle class used in FMA loader and writer.

The pieces of data are : PosX, PosY, PosZ, physicalValue, Potential, forceX, forceY, forceZ. The first 4 are mandatory. Data is stored as FReal.

See FFmaGenericLoader for information on the format.

模板参数
READnumber of items to read (UNUSED, see WRITE)
WRITEnumber of items to write (must be >= 4)
