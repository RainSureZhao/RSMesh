\doxysubsection{详细描述}
This class define the methods that every cell class has to implement to be insertable in the tree. 

\begin{DoxyAuthor}{作者}
Berenger Bramas (\href{mailto:berenger.bramas@inria.fr}{\texttt{ berenger.\+bramas@inria.\+fr}})
\end{DoxyAuthor}


In fact \doxylink{class_f_octree}{FOctree} \& \doxylink{class_f_fmm_algorithm}{FFmm\+Algorithm} need this functions to be implemented. But you cannot use this interface with the extension (as an example \+: because the compiler will faill to know if get\+Morton\+Index is coming from this interface or from the extension)

You can look to \doxylink{class_f_basic_cell}{FBasic\+Cell} to have an idea of implementation.
