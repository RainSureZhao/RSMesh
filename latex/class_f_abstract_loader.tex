A loader is the component that fills an octree.

If you want to use a specific file format you then need to inherit from this loader and implement several methods.

Please look at \doxylink{class_f_basic_loader}{FBasic\+Loader} or FFma\+Loader to see an example.

\begin{DoxyWarning}{警告}
Inherit from this class when defining a loader class 
\end{DoxyWarning}
