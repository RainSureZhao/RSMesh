This class defines a basic cell used for examples. It extends the mininum, only what is needed by \doxylink{class_f_octree}{FOctree} and \doxylink{class_f_fmm_algorithm}{FFmm\+Algorithm} to make the things working. By using this extension it will implement the \doxylink{class_f_abstract_cell}{FAbstract\+Cell} without inheriting from it.

Berenger Bramas (\href{mailto:berenger.bramas@inria.fr}{\texttt{ berenger.\+bramas@inria.\+fr}})
\end{DoxyAuthor}


This cell simply store the data when up/down. It also shows how to be restored and saved, etc.
