\subsubsection*{template$<$class FReal, unsigned Nb\+Attributes\+Per\+Particle, class Attribute\+Class$>$\newline
class FBasic\+Particle\+Container$<$ FReal, Nb\+Attributes\+Per\+Particle, Attribute\+Class $>$}
\begin{DoxyAuthor}{作者}
Berenger Bramas (\href{mailto:berenger.bramas@inria.fr}{\texttt{ berenger.\+bramas@inria.\+fr}}) Please read the license
\end{DoxyAuthor}
This class defines a container which can holds one type (Attribute\+Class) for each particle. The memory is allocated in one block and stores the positions and the request type. For example if one wants to store a struct for each particle\+: 
\begin{DoxyCode}{0}
\DoxyCodeLine{\textcolor{keyword}{struct\ }AStruct\{\ ...\ \};}
\DoxyCodeLine{\mbox{\hyperlink{class_f_basic_particle_container}{FBasicParticleContainer<FReal,\ 1,\ AStruct>}}\ container;}

\end{DoxyCode}
 And then access is done using\+: 
\begin{DoxyCode}{0}
\DoxyCodeLine{AStruct*\ strucs\ =\ container.getAttributes<0>();}

\end{DoxyCode}
 For example if one wants to store 4 doubles for each particles\+: 
\begin{DoxyCode}{0}
\DoxyCodeLine{\mbox{\hyperlink{class_f_basic_particle_container}{FBasicParticleContainer<FReal,\ 4,\ double>}}\ container;}

\end{DoxyCode}
 And then access is done using\+: 
\begin{DoxyCode}{0}
\DoxyCodeLine{\textcolor{keywordtype}{double}*\ v1\ =\ container.getAttributes<0>();}
\DoxyCodeLine{\textcolor{keywordtype}{double}*\ v2\ =\ container.getAttributes<1>();}
\DoxyCodeLine{\textcolor{keywordtype}{double}*\ v3\ =\ container.getAttributes<2>();}
\DoxyCodeLine{\textcolor{keywordtype}{double}*\ v4\ =\ container.getAttributes<3>();}

\end{DoxyCode}
 The memory is aligned to FP2\+PDefault\+Alignement value.
