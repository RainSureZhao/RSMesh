This kernels implement the Chebyshev interpolation based FMM operators exploiting the symmetries in the far-\/field. It implements all interfaces (P2P, P2M, M2M, M2L, L2L, L2P) which are required by the \doxylink{class_f_fmm_algorithm}{FFmm\+Algorithm} and \doxylink{class_f_fmm_algorithm_thread}{FFmm\+Algorithm\+Thread}.
