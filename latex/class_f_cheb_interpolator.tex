The class {\ttfamily \doxylink{class_f_cheb_interpolator}{FCheb\+Interpolator}} defines the anterpolation (M2M) and interpolation (L2L) concerning operations.

PB\+: Matrix\+Kernel\+Class is passed as template in order to inform interpolator of the size of the vectorial interpolators. Default is the scalar matrix kernel class of type ONE\+\_\+\+OVER\+\_\+R (NRHS=NLHS=1).
