This kernels implement the Chebyshev interpolation based FMM operators. It implements all interfaces (P2P, P2M, M2M, M2L, L2L, L2P) which are required by the \doxylink{class_f_fmm_algorithm}{FFmm\+Algorithm} and \doxylink{class_f_fmm_algorithm_thread}{FFmm\+Algorithm\+Thread}.

\hline
{\em Container\+Class} & Type of container to store particles \\
\hline
{\em Matrix\+Kernel\+Class} & Type of matrix kernel function \\
\hline
{\em ORDER} & Chebyshev interpolation order \\
\hline
\end{DoxyTemplParams}
