\doxysubsection{详细描述}
Time counter class. 

\begin{DoxyAuthor}{作者}
Berenger Bramas (\href{mailto:berenger.bramas@inria.fr}{\texttt{ berenger.\+bramas@inria.\+fr}})
\end{DoxyAuthor}
This time counter can be (re)started using \doxylink{class_f_cuda_tic_a3f7d75cb23e64b344bbdb81c0fb713f5}{tic()} and stopped using \doxylink{class_f_cuda_tic_a32e8d471c975e48a4b5cb1a27ec6afaa}{tac()}.


\begin{DoxyItemize}
\item use \doxylink{class_f_cuda_tic_ade8f59e6471dc102487ef175335d0a3a}{elapsed()} to get the last time interval;
\item use \doxylink{class_f_cuda_tic_ac6d3a8f744024d18e7c183399e8375dd}{cumulated()} to get the total running time;
\item use \doxylink{class_f_cuda_tic_af3423c417c074373da0d78d9d5ab0abe}{reset()} to stop and reset the counter.
\end{DoxyItemize}
