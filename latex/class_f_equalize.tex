\doxysubsection{详细描述}
This class proposes a method to distribute an interval own by a worker to some others. It returns a vector of \doxylink{struct_f_equalize_1_1_package}{Package} which tell what interval to sent to which process. The algorithm works only if the objectives (the intervals that the workers shoud obtain are exclusive and given in ascendant order). Also each of the element from the current interval must belong to someone! Finally the current worker is included in the \doxylink{struct_f_equalize_1_1_package}{Package} if its objective interval is related to its starting interval.
