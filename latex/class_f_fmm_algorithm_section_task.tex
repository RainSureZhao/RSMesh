Parallel FMM algorithm implementation using Open\+MP sections. Please read the license 

\begin{DoxyAuthor}{作者}
Berenger Bramas (\href{mailto:berenger.bramas@inria.fr}{\texttt{ berenger.\+bramas@inria.\+fr}})
\end{DoxyAuthor}


This class implements the FMM algorithm. the far field and near field computations are parallelized using Open\+MP sections (the L2P operator is done last)

\begin{DoxyWarning}{警告}
Because of the parallelization in this class, the only timers that are meaningfull after a call to \doxylink{class_f_fmm_algorithm_section_task_a9f8868b83e97f0eca0f7187903f6064a}{execute\+Core} are the P2M timer, which times the far (P2M,M2M,M2L,L2L) and near (P2P) field computation and the Near\+Timer (L2P).
\end{DoxyWarning}
Upon destruction, this class does not deallocate pointers given to its constructor.
