class FGroup\+Attached\+Leaf$<$ FReal, Nb\+Symb\+Attributes, Nb\+Attributes\+Per\+Particle, Attribute\+Class $>$}
This class is "{}attached"{} to a buffer. It is a wrapper/interface to a memory area that stores all the particles info. The idea is to hidde the group allocation from the group tree but to keep the same interface than the FBasic\+Particles\+Container.
