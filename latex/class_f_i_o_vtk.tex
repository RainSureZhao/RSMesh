Print the graph in a VTK file 

The output file name is \mbox{[}VTKName\mbox{]}\mbox{[}ifile\mbox{]}.vtk where ndigits specifies the number of characters to be written for \mbox{[}ifile\mbox{]}.

We save in VTK 3.\+0 format and in ascii mode. See for more details on this format

\href{http://www.cacr.caltech.edu/~slombey/asci/vtk/vtk_formats.simple.html}{\texttt{ http\+://www.\+cacr.\+caltech.\+edu/\texorpdfstring{$\sim$}{\string~}slombey/asci/vtk/vtk\+\_\+formats.\+simple.\+html}}

The data saved are the following.
\begin{DoxyItemize}
\item On Nodes\+: velocitie and force.
\item On segments\+: length, burgers vector and morton index.
\end{DoxyItemize}
