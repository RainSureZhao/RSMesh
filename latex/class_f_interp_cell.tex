\subsubsection*{template$<$class FReal, int ORDER, int NRHS = 1, int NLHS = 1, int NVALS = 1$>$\newline
class FInterp\+Cell$<$ FReal, ORDER, NRHS, NLHS, NVALS $>$}
\begin{DoxyAuthor}{作者}
Pierre Blanchard (\href{mailto:pierre.blanchard@inria.fr}{\texttt{ pierre.\+blanchard@inria.\+fr}})
\end{DoxyAuthor}
Please read the license

This class defines a cell that can be used in both Chebyshev and Lagrange based FMM.

PB\+: This class provides the storage and accessors for the transformed expansion used in the Uniform FMM (in Fourier space, i.\+e. complex valued).

PB\+: Beware! This class does not support yet the storage of the compressed expansions used in the low rank version of the Chebyshev FMM algorithm.


\begin{DoxyParams}{参数}
{\em NVALS} & is the number of right hand side. \\
