class FOctree\+Arranger\+Proc$<$ FReal, Octree\+Class, Container\+Class, Particle\+Class, Converter\+Class $>$}
This class is an arranger, it move the particles that need to be hosted in a different leaf. This is the parallel version that use MPI. 

For example, if a simulation has been executed and the position of the particles have been changed, then it may be better to move the particles in the tree instead of building a new tree.
