class FP2\+PExclusion$<$ Box\+Separations $>$}
This class gives is responsible of the separation of the leaves using the coloring algorithm. In case of classic P2P and mutual interaction the Box\+Separations = 2 should be used. For our current mutual P2P is is a little more complicated because we need 2 boxes of separation but only in some directions.
