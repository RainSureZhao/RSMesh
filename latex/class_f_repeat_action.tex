\doxysubsection{详细描述}
Calls a function at time interval until stopped. 

This object will call at a defined interval any function or lambda that is passed to it. The function can stop the call by returning false.

For convenience, the class inherits from \doxylink{class_f_tic}{FTic} to measure time.
