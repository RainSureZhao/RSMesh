\subsubsection*{template$<$class FReal, int P$>$\newline
class FRotation\+Cell$<$ FReal, P $>$}
This class is a cell used for the rotation based kernel The size of the multipole and local vector are based on a template User should choose this parameter P carrefuly to match with the P of the kernel.

Multipole/\+Local vectors contain value as\+: \{0,0\}\{1,0\}\{1,1\}...\{P,P-\/1\}\{P,P\} So the size of such vector can be obtained by a suite\+: (n+1)\texorpdfstring{$\ast$}{*}n/2 =\texorpdfstring{$>$}{>} (P+2)\texorpdfstring{$\ast$}{*}(P+1)/2
