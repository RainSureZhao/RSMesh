\doxysubsection{详细描述}
Time counter class. 

\begin{DoxyAuthor}{作者}
Berenger Bramas (\href{mailto:berenger.bramas@inria.fr}{\texttt{ berenger.\+bramas@inria.\+fr}})
\end{DoxyAuthor}
This time counter can be (re)started using \doxylink{class_f_tic_ac0e070e65af128fbe5af711e51033b0a}{tic()} and stopped using \doxylink{class_f_tic_a3efff71f7c95acd38451bfab08fa4583}{tac()}.


\begin{DoxyItemize}
\item use \doxylink{class_f_tic_a30b9f876fb7188f85b4c2b9d7fd7943e}{elapsed()} to get the last time interval;
\item use \doxylink{class_f_tic_ac53ed8191e3b5d9dadb88cae3f00a55e}{cumulated()} to get the total running time;
\item use \doxylink{class_f_tic_add8f7cf5d655cb3858f1b93ee6d715a5}{reset()} to stop and reset the counter.
\end{DoxyItemize}
