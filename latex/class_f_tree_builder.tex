This class provides a way to insert efficiently large amount of particles inside a tree.

This is a static class. It\textquotesingle{}s useless to instance it. This class use the Threaded Quick\+Sort or the output of \doxylink{class_f_mpi_tree_builder}{FMpi\+Tree\+Builder} in order to sort the parts and insert them.
