\begin{DoxyAuthor}{作者}
Pierre Blanchard (\href{mailto:pierre.blanchard@inria.fr}{\texttt{ pierre.\+blanchard@inria.\+fr}})
\end{DoxyAuthor}


This kernels implement the Lagrange interpolation based FMM operators. It implements all interfaces (P2P,P2M,M2M,M2L,L2L,L2P) which are required by the \doxylink{class_f_fmm_algorithm}{FFmm\+Algorithm} and \doxylink{class_f_fmm_algorithm_thread}{FFmm\+Algorithm\+Thread}. This is the dense version of the kernel. The transfer are done in real space and not in Fourier space.

\hline
