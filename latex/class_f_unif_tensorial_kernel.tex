This kernels implement the Lagrange interpolation based FMM operators. It implements all interfaces (P2P,P2M,M2M,M2L,L2L,L2P) which are required by the \doxylink{class_f_fmm_algorithm}{FFmm\+Algorithm} and \doxylink{class_f_fmm_algorithm_thread}{FFmm\+Algorithm\+Thread}.

PB\+: 3 IMPORTANT remarks !!!

1) Handling tensorial kernels (DIM,NRHS,NLHS) and having multiple rhs (NVALS) are considered 2 distinct features and are currently combined.

2) When it comes to applying M2L it is NOT much faster to loop over NRHSx\+NLHS inside apply\+M2L (at least for the Lagrange case). 2-\/bis) During precomputation the tensorial matrix kernels are evaluated blockwise, but this is not always possible. In fact, in the Chebyshev\+Sym variant the matrix kernel needs to be evaluated compo-\/by-\/compo since we currently use a scalar ACA.

3) We currently use multiple 1D FFT instead of multidim FFT since embedding is circulant. Multidim FFT could be used if embedding were block circulant. TODO investigate possibility of block circulant embedding
