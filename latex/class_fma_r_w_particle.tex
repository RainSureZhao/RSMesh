\subsubsection*{template$<$class FReal, unsigned int READ, unsigned int WRITE$>$\newline
class Fma\+RWParticle$<$ FReal, READ, WRITE $>$}
Particle class used in FMA loader and writer. 

The pieces of data are \+: PosX, PosY, PosZ, physical\+Value, Potential, forceX, forceY, forceZ. The first 4 are mandatory. Data is stored as FReal.

See \doxylink{class_f_fma_generic_loader}{FFma\+Generic\+Loader} for information on the format.
