Deals with all the symmetries in the arrangement of the far-\/field interactions 

Stores permutation indices and permutation vectors to reduce 316 (7\texorpdfstring{$^\wedge$}{\string^}3-\/3\texorpdfstring{$^\wedge$}{\string^}3) different far-\/field interactions to 16 only. We use the number 343 (7\texorpdfstring{$^\wedge$}{\string^}3) because it allows us to use to associate the far-\/field interactions based on the index $t = 7^2(i+3) + 7(j+3) + (k+3)$ where $(i,j,k)$ denotes the relative position of the source cell to the target cell.
